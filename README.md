# Todolist

### 建立虛擬環境
- pip install pipenv
- pip shell

### 安裝套件
- pip install django

### 產生專案
- django-admin startproject todolist .
    - . => 本地產生需求的目錄跟檔按

### 啟動Server
- python manage.py runserver


### 同步資料庫跟建立資料表
- python manage.py makemigrations
- python manage.py migrate

# 建立超級使用者
- python manage.py createsuperuser
